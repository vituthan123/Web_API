### DOCKER
- Turn on docker
- Open and run in CMD:
    `docker-compose up -d`
- Turn off docker
    `docker-compose down --rmi all`
- Craw data:
    `localhost:5556/docs -All Method`
    `localhost:5556/CrawData - Lấy tổng hợp 3 dữ liệu dưới `
- Save to DB:
    `localhost:9000/docs -All Method`
    `localhost:9000/CrawToDB - Lưu tất cả dữ liệu vào DB`
- Web_Search:
    `localhost:8080`
